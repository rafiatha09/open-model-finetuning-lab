# Phase 2: Open-Model Usage

## Goal

Learn to run one or more open models locally or remotely before training.

## What to build here

- one baseline inference script
- one prompt template for the target domain assistant
- notes on model choice tradeoffs:
  size, latency, license, context length, and instruction-following quality
